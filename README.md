# Projeto de Técnicas de Programação – Adatech / CaixaVerso
Este repositório contém um projeto desenvolvido como parte do curso de Técnicas de Programação promovido pela Adatech em parceria com a Caixa.

## Objetivo do projeto
Realizar o tratamento e a análise exploratória de dados utilizando a biblioteca Pandas, a partir de um dataset público relacionado ao consumo de energia.

## Dataset
Os dados utilizados no projeto referem-se a informações sobre consumo e fontes de energia, com o objetivo de identificar padrões, comparações entre países/regiões e possíveis tendências ao longo do tempo.

O dataset encontra-se disponível neste repositório em formato CSV.

## Ferramentas utilizadas
- Python
- Pandas
- Google Colab
- GitHub
- Copilot

## Estrutura do repositório
- arquivo CSV com os dados utilizados
- notebook (.ipynb) com a análise
- arquivo README.md com a descrição do projeto

## Observações
Este projeto tem caráter educacional e foi desenvolvido para fins de aprendizado em análise de dados.
